  • iOS Development

    When it comes to my main focus, this is it. I have dedicated the majority of my time studying and applying concepts to develop and produce high quality mobile iOS applications. These include SwiftUI, MVC, MVVM, CocoaPods, TableView, URLSession, JSON parsing, Firebase, UIKit, RESTAPIs, and CoreData just to name a few. iOS is my favorite OS and being able to develop natively has allowed me to not only explore career opportunities but to release my own app in the App Store coming mid 2021!

  • Flutter Development

    My journey into mobile development starts here. I discovered Flutter in early 2020 and have had a blast developing using the framework. A tool to use the same code base for iOS and Android seemed very powerful and has fast-tracked my understanding of mobile development practices. Bloc seems like the go-to for state management and I have committed to that method for production. I tend to use Flutter when working with the Andriod SDK but I have used it to manage plist files in iOS builds.
